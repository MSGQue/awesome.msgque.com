What is MSGQue

MSGQue is an short messaging service server built for cloud application developers.



Kannel Compatible

MSGQue built compatiable with Kannel. MSGQue also gives seemless experience.

Written in Node.js

Node.js Asynchronous event driven IO helps MSGQue to handle concurrent requests.

Apache Licence 2.0

MSGQue is free software, released under Apache license v2.0. MSGQue has an active developer and user community.
